# Hacker News clone

A personal scaffold with following libraries/tools/functionalities
- react, redux, react-router
- styled-components
- lerna, webpack, babel
- jest, eslint
- universal javascript, server-side rendering
- hot reload on client and server side

## Installation
```
npm install
npm run bs
```

## Run
```
npm build
npm start
```

## Development
```
npm run dev
```
